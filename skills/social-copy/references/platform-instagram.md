# Instagram Platform Guide

**Best for:** Visual brands, lifestyle, younger demographics
**Optimal frequency:** 3-5x/week + stories daily
**Best format:** Carousels (6.9% engagement), Reels (2.25x reach vs images)

## What works
- Reels (short-form video prioritized by algorithm)
- Carousels with educational value (10 slides)
- Behind-the-scenes Stories
- User-generated content
- Interactive Stories (polls, questions, sliders)

## Caption structure
```
Hook (first line visible before "more")
Line break
Body (value/educational content)
Line break
CTA (save, share, comment, link in bio)
Line break
Hashtags (3-5 relevant, 1 branded)
```

## Reels structure
```
0-2s: Hook (visual + text overlay)
2-10s: Context/setup
10-25s: Value delivery
25-30s: CTA
```

## Algorithm tips
- Reels prioritized over static posts
- Saves and shares > likes
- First frame of Reels must hook
- Use trending audio when relevant

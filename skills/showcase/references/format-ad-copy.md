# Ad Copy Format

Headline + body + CTA generation. Zero dependencies.

## Structure

```
Headline (30-40 chars): Direct benefit or curiosity gap
Body (90-125 chars): Problem → solution → proof
CTA (2-5 words): Strong verb + clear outcome
```

## Headline Formulas

| Formula | Example |
|---------|---------|
| [Benefit] + [Specific result] | "Build products. Ship promotion." |
| How to [achieve X] | "How to launch without marketing stress" |
| [Number] ways to [goal] | "One describe. Five formats." |
| Question + pain | "Built something nobody sees?" |

## Platform Limits

| Platform | Headline | Body | CTA |
|----------|----------|------|-----|
| Google | 30 chars | 90 chars | — |
| Facebook | 40 chars | 125 chars | 25 chars |
| LinkedIn | 70 chars | 150 chars | 25 chars |

## Frameworks

- **AIDA:** Attention → Interest → Desire → Action
- **PAS:** Problem → Agitate → Solve
- **FAB:** Feature → Advantage → Benefit

# Quality Gates

Per-format verification checklist. Run after generation, before distribution.

## Universal (all formats)

- [ ] Brand voice compliance — no vocabulary drift from `product-marketing.md`
- [ ] No generic SaaS language — "streamline", "elevate", "seamless", "revolutionize" banned
- [ ] CTA present and specific (not "learn more" without context)
- [ ] Output feels made for this product, not any product
- [ ] Reading time appropriate for format

## Social Post

- [ ] Character limits: LinkedIn (3000), Twitter (280), IG caption (2200)
- [ ] Line breaks for scannability
- [ ] Hashtag count: 3-5 relevant, not stuffed

## Ad Copy

- [ ] Headline: 30-40 chars
- [ ] Body: 90-125 chars
- [ ] CTA: 2-5 words, strong verb
- [ ] Uses AIDA/PAS/FAB framework

## Video

- [ ] Duration: 15-25s
- [ ] Hook in first 2 seconds
- [ ] Shows actual product UI or visual
- [ ] Text on screen readable (0.3s per word minimum)
- [ ] Music + SFX if appropriate

## Carousel

- [ ] N slides match plan
- [ ] One idea per slide (not more)
- [ ] Slide 1 is hook (not logo/title intro)
- [ ] Last slide is CTA
- [ ] Text readable on mobile

## Reel

- [ ] Vertical (9:16)
- [ ] Hook in first 2s (visual + text overlay)
- [ ] Text overlay: 2-4 words per frame
- [ ] Platform-specific duration respected

## Scorig (0-4 per dimension)

| Dimension | 0 | 1 | 2 | 3 | 4 |
|-----------|---|---|---|---|---|
| Brand Voice | Off-brand | Some drift | Neutral | On-brand | Perfect |
| Platform Rules | Violated | Minor issues | Compliant | Optimized | Exemplary |
| Persuasion | Missing | Weak | Present | Strong | Compelling |
| Readability | Unreadable | Dense | Fine | Good | Excellent |
| AI Slop | Full slop | Some slop | Neutral | Specific | Unique |

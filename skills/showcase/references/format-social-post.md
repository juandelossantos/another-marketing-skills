# Social Post Format

Platform-specific copy generation. Zero dependencies — pure text output.

## LinkedIn

**Structure:** Hook (bold claim) → Story/context → Insight → CTA
**Length:** 150-300 words. Line breaks between sections.
**Tone:** Professional but conversational. Insight-driven.

## Twitter/X

**Structure:** Hook → Value → CTA
**Length:** 280 chars or thread (3-5 tweets)
**Tone:** Sharp, quick. Curiosity gaps work well.

## Instagram

**Structure:** Hook (first line visible) → Body → CTA → Hashtags (3-5)
**Length:** 100-200 words caption.
**Tone:** Visual-first, casual, inspirational.

## Quality checks

- [ ] Character limits respected per platform
- [ ] CTA present (link, follow, comment, save)
- [ ] Tone matches platform expectation
- [ ] Hashtag strategy per platform

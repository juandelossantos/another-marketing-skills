# Reel Format (Short-Form Vertical)

Script for TikTok, Instagram Reels, YouTube Shorts. Requires HEYGEN_API_KEY for rendering.

## Structure (15-30s)

```
Hook (0-2s) → Context (2-6s) → Payoff (6-20s) → CTA (20-30s)
```

## Vertical specs

- Aspect ratio: 9:16
- Resolution: 1080x1920
- Text safe zone: center 2/3 of screen
- Text overlay: 2-4 words per frame (reading time ~2.5 words/s)

## Platform notes

| Platform | Max duration | Hashtags | Caption style |
|----------|-------------|----------|---------------|
| TikTok | 60s (ideally 15-30) | 3-5 relevant | Blog-style 150+ words |
| Reels | 90s | 3-5 relevant | Short, punchy |
| Shorts | 60s | 3-5 relevant | Description with keywords |

## Quality gates

- [ ] Duration within platform limit
- [ ] Hook in first 2 seconds (visual + text)
- [ ] Text overlay readable (high contrast, 2-4 words)
- [ ] CTA at end (follow, save, comment)
- [ ] Vertical format (9:16)

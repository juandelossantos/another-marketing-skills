# Meta Tags Guide

Source: Google SEO Starter Guide, Schema.org.

## Title Tag
- Length: 50-60 characters (longer gets truncated in SERP)
- Primary keyword near the beginning
- Brand name at end (separated by "—" or "|")
- Unique per page
- No keyword stuffing
- Example: `"AI Marketing Skills — Promote your product in 5 formats"`

## Meta Description
- Length: 150-160 characters
- Includes primary keyword naturally
- Clear value proposition
- Call to action ("Learn more", "Get started", "Try it")
- Unique per page
- No quotation marks (breaks at SERP display)
- Example: `"Generate video, social posts, email sequences, carousels, and ads from one product description. Open source. MIT."`

## Viewport
```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

## Robots
```html
<meta name="robots" content="index, follow">
```

## Canonical
```html
<link rel="canonical" href="https://example.com/page/">
```

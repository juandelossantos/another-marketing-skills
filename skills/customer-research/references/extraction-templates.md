# Extraction Templates

## Source Log Entry

```markdown
## Source: [Platform] — [Title/Thread]
**URL:** [link]
**Date:** [date]
**Segment hint:** [role, company size, industry if detectable]

### Verbatim Quote
"> [exact quote]"

### Context
[What prompted this comment? What's the situation?]

### Category
[Pain / Trigger / Outcome / Language / Alternative]

### Sentiment
[Positive / Negative / Frustrated / Neutral]
```

## Synthesis Report Template

```markdown
# Research Synthesis

## Top Themes (ranked by frequency × intensity)

### Theme 1: [Name]
**Summary:** [1-2 sentences]
**Frequency:** X of Y sources
**Intensity:** High / Medium / Low
**Representative quotes:**
- "[quote]" — [source, date]
- "[quote]" — [source, date]
**Implications:** [What this means for messaging / product / positioning]

### Theme 2: ...
```

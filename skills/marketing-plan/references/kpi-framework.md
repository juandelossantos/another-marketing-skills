# KPI Framework

## North Star Metric
The single metric that indicates long-term success.
Example for devtools: "Active projects using our tool"

## Leading Indicators (by AARRR stage)
| Stage | Leading indicator | Lagging indicator |
|-------|------------------|-------------------|
| Acquisition | Website traffic, social reach | Signups |
| Activation | Onboarding completion % | 7-day active users |
| Retention | Email open rate, feature usage | Churn rate |
| Referral | NPS, sharing rate | Referral signups |
| Revenue | Trial-to-paid conversion | MRR, ARR |

## Quarterly Targets
Set specific numbers per KPI. Review monthly, pivot quarterly.
Minimum 3 months to show real impact from any new channel.

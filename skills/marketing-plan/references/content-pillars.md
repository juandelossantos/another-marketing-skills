# Content Pillars

## Pillar-Cluster Model
1 pillar page (comprehensive guide) → 10-20 cluster pages (specific subtopics)

## Recommended Split
| Pillar type | % | Examples |
|-------------|---|----------|
| Industry insights | 30% | Trends, data, analysis |
| Educational | 25% | How-tos, tutorials, frameworks |
| Behind-the-scenes | 25% | Building process, lessons |
| Personal/story | 15% | Founder stories, values |
| Promotional | 5% | Product updates, launches |

## Repurposing Ratio
1 pillar content → 5-8 repurposed pieces
Prioritize formats where audience is most active.

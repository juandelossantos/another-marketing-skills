# Onboarding Sequence (5-7 emails, 14 days)

Goal: Drive users to aha moment, activate key behaviors, upgrade.

## Email 1 — Welcome + First Step
Send: Immediately
Subject: Let's set up your [product] account
Body: First action they need to take + expected time
CTA: Complete setup

## Email 2 — Getting Started Help
Send: Day 1
Subject: 3 things to do in your first week
Body: Prioritized list of setup tasks
CTA: Do step 1

## Email 3 — Feature Highlight
Send: Day 2-3
Subject: The feature that changes everything
Body: Show one powerful feature with example
CTA: Try [feature]

## Email 4 — Success Story
Send: Day 4-5
Subject: How another user achieved [result]
Body: Relatable use case from similar customer
CTA: Learn from them

## Email 5 — Check-in
Send: Day 7
Subject: How's it going? (really)
Body: Genuine check-in + support resources
CTA: Reply with questions

## Email 6 — Advanced Tip
Send: Day 10-12
Subject: A pro tip for [product] users
Body: Advanced usage that unlocks more value
CTA: Try this pro tip

## Email 7 — Upgrade
Send: Day 14
Subject: You've outgrown the free plan?
Body: Recap value + what premium unlocks
CTA: Upgrade now
